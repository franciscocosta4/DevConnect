# DevConnect – Plataforma Social para Programadores

Aplicação web desenvolvida com **Laravel** e **MySQL** que funciona como uma mini-rede social para programadores — uma combinação conceptual entre GitHub e Twitter, focada na partilha de conteúdo técnico e interacção entre developers.

O objectivo do projecto é demonstrar competências de desenvolvimento full-stack com arquitectura MVC, autenticação, base de dados relacional e interface moderna.

## Instalação Local

### 1. Clonar o repositório

Clona o projecto para a tua máquina local através do Git.

### 2. Configurar ambiente

- Criar um ficheiro `.env` com base no `.env.example`
- Configurar as credenciais da base de dados MySQL

### 3. Instalar dependências

Instalar dependências PHP com Composer  
Instalar dependências front-end com NPM  

### 4. Gerar chave da aplicação

Executar o comando artisan para gerar a chave da aplicação.

### 5. Migrar base de dados

Executar as migrations para criar as tabelas necessárias no MySQL.

### 6. Iniciar servidor

Iniciar o servidor de desenvolvimento Laravel e o processo de compilação de assets.

A aplicação ficará disponível em ambiente local.
